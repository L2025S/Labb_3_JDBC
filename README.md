Features<br>
• Load database configuration from `application.properties• Connect to a relational database using JDBC<br>
• CRUD operations on a person table: <br>
	◦ Insert a new person <br>
	◦ Update an existing person <br>
	◦ Delete a person by ID <br>
	◦ Find a person by ID <br>
	◦ Retrieve all persons <br>
• A service layer that performs validation before calling the DAO  <br>

Technologies Used   <br>
• Java   <br>
• JDBC   <br>
• SQL   <br>
• DAO design pattern   <br>
• Service layer validation   <br>
